# Quick Secure Market App Web

Plataforma inteligente de comercio electrónico seguro enfocada en la prevención de fraudes digitales en República Dominicana.

---

## Tecnologías

- React
- Vite
- Node.js
- Express
- MongoDB
- Tailwind CSS

---

## Arquitectura

### Backend

```
backend/
    controllers/
    middleware/
    models/
    routes/
    server.js
```

### Frontend

```
frontend/
    src/
        components/
        pages/
        assets/
        services/
        App.jsx
        main.jsx
```

---

## Módulos principales

- Login y Registro
- Dashboard
- Marketplace
- Perfil
- Publicar productos
- Carrito
- Chat Messenger
- Escrow
- Disputas
- IA Antifraude
- Marketing Center
- Panel Administrativo

---

## Objetivo

Crear la plataforma de comercio electrónico más segura de República Dominicana mediante:

- Validación biométrica
- Verificación de identidad
- Escrow de pagos
- Inteligencia artificial antifraude
- Sistema de reputación y disputas
